# homebrew-tele

Homebrew tap for [tele](https://github.com/agmalpartida/tele) — keyboard-driven TUI Telegram client.

## Install

```sh
brew tap agmalpartida/tele
brew install tele
```

## Upgrade

```sh
brew upgrade tele
```

## Uninstall

```sh
brew uninstall tele
brew untap agmalpartida/tele
```
